# ============================================================================
# DEPENDENCY RESOLUTION NOTES
# ============================================================================
# 
# WHY THESE SPECIFIC VERSIONS?
# 
# 1. NUMPY CONFLICT RESOLUTION:
#    - Streamlit 1.28.1 requires: numpy < 2
#    - langchain-community 0.1.0+ requires: numpy >= 2.1.0
#    - SOLUTION: Use langchain-community 0.0.38 (last v0.0.x version)
#    - SOLUTION: Pin numpy to 1.26.4 (latest stable v1)
#
# 2. LANGCHAIN VERSION STRATEGY:
#    - langchain 1.0.0+ has breaking changes and numpy conflicts
#    - langchain 0.1.14 is stable and compatible with all components
#    - All langchain-* packages pinned to compatible versions
#
# 3. STREAMLIT COMPATIBILITY:
#    - Streamlit 1.28.1 is the latest stable version for this setup
#    - Streamlit 1.29+ requires numpy >= 2 (incompatible)
#    - Streamlit 1.28.1 works perfectly with numpy 1.26.4
#
# 4. CHROMADB STABILITY:
#    - ChromaDB 0.4.24 is stable and well-tested
#    - Newer versions may have numpy conflicts
#    - 0.4.24 works reliably with all dependencies
#
# 5. SENTENCE TRANSFORMERS:
#    - Version 2.2.2 is stable and compatible
#    - Version 2.3+ may introduce numpy conflicts
#    - 2.2.2 works with numpy 1.26.4
#
# 6. PYTHON VERSION:
#    - Python 3.11 is recommended (not 3.14)
#    - Python 3.14 is experimental and may have compatibility issues
#    - All packages tested with Python 3.11
